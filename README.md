# Amazon-clone
This is an exact front-end replica of Amazon.com using HTML and CSS.<br><br>

<b>Amazon Homepage Clone</b>

A static Amazon homepage clone built using HTML5 and CSS3, focusing on layout design, Flexbox alignment, and responsive UI components.
This project is created for learning and practice purposes only.<br><br>

<b>Features:</b>

  ->Amazon-style navigation bar
  
  ->Location selector and search bar
  
  ->Product categories section
  
  ->Hero banner section
  
  ->Responsive shop grid layout
  
  ->Fully designed footer section
  
  ->Font Awesome icons integration<br><br>

<b>Technologies Used</b>

  ->HTML5
  
  ->CSS3
  
  ->Flexbox
  
  ->Font Awesome (CDN)<br><br>
